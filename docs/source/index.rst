Welcome to feqab-messenger-bot documentation!
===================================

**feqab-messenger-bot** is an all in one Facebook Messenger Bot with tons of commands

It pulls data from the `frenzvalios API <https://frenzvalios.tk/api/>`_
and offers a *simple* and *intuitive* API.

Check out the :doc:`usage` section for further information, including
how to :ref:`add to GC` the Bot.

.. note::

   This project is under active development.
